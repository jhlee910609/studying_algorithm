# Sorting(정렬)

## 1. 정렬의 개념
- 순서없이 배열되어 있는 자료들을 작은 것부터 큰 것 순서의 오름차순이나 내림차순으로 재배열하는 것


> 숫자들의 min, max 값 구하기, 문자열 재배열 (아스키코드 활용)

- 자료를 정렬하는 데 사용하는 기준이 되는 특정 값을 **키(`key`)**라고 부름
